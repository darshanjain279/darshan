Intervals

Not started yet — 7 problems queued.

Sort by start (or end) time first — nearly every interval problem begins there. Then sweep left to right, merging or comparing consecutive intervals.

Sub-patterns

  • A — Merge Overlapping. Collapse a list of intervals into their disjoint union.
  • B — Insert a New Interval. Add one interval into an already-sorted, disjoint list.
  • C — Overlap Counting. Count how many intervals overlap at any point.
  • D — Interval Intersection. Find the overlap between two lists of intervals.
